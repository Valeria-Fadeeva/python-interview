complex() создает комплексное число:
complex(3.5,4)
(3.5+4j)

eval() исполняет строку:
eval("print(max(22,22.0) — min(2,3))")
20

filter() отфильтровывает элементы, для которых заданное условие верно
list(filter(lambda x: x%2 == 0,[1, 2, 0, False]))
[2, 0, False]

format() помогает задать формат строки:
print("a = {0} but b = {1}".format(a, b))
a = 2 but b = 3

hash() возвращает хэш-значение объекта:
hash(3.7)
644245917

hex() преобразовывает число в шестнадцатеричное число:
hex(14)
"0xe"

input() читает ввод и возвращает строку:
input("Enter a number")
Enter a number7
"7"

len() возвращает число, показывающее длину строки:
len("Ayushi")
6

locals() возвращает словарь с локальной таблицей имен:
locals()

open() открывает файл:
file = open("tabs.txt")
